# Mojito Cocktail Landing Page

## Overview

* **Mojito Cocktail Landing Page** is a modern and interactive single-page website designed to showcase the refreshing experience of a classic Mojito cocktail. The project focuses on smooth animations, responsive layouts, and immersive visual storytelling to create an engaging user experience.

* Built with React and powered by GSAP, the landing page delivers fluid transitions, scroll-based animations, and a premium interface optimized for both desktop and mobile devices.

**🗓️ Project created on:** June 24, 2026

## Technologies & Libraries

* **ReactJS:** Builds a fast and component-based user interface.
* **GSAP:** Creates high-performance animations and timeline sequences.
* **@gsap/react:** Integrates GSAP seamlessly with React components.
* **ScrollTrigger:** Implements scroll-driven animations and effects.
* **SplitText:** Generates character, word, and line-based text animations.
* **Tailwind CSS:** Provides a utility-first styling system for rapid UI development.
* **react-responsive:** Detects screen sizes for adaptive layouts and media experiences.
